# Arithmetic Operators
| Operator | Description |
|----------|-------------|
| `+`      | Addition |
| `-`      | Subtraction |
| `*`      | Multiplication |
| `/`      | Division |
| `%`      | Modulus (integers only) |
| `++`     | Increment |
| `--`     | Decrement |

**Notes:**
- Operate on built-in types or overloaded operators.
- Pre/post-increment have different evaluation order.
