# C++ Cast Operators
| Operator | Description |
|----------|-------------|
| `static_cast<T>(expr)` | Compile-time checked conversion |
| `dynamic_cast<T>(expr)` | Runtime-checked polymorphic cast |
| `const_cast<T>(expr)` | Remove/add const/volatile qualifiers |
| `reinterpret_cast<T>(expr)` | Bitwise reinterpretation |
