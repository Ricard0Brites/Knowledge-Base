# Miscellaneous Operators
| Operator | Description |
|----------|-------------|
| `?:`     | Ternary conditional |
| `sizeof` | Size of type or object |
| `typeid` | Type information at runtime |
| `,`      | Comma operator, evaluates left then right |
| `.`      | Member access |
| `->*`    | Pointer-to-member access |
| `.*`     | Member access via object |
| `::`     | Scope resolution |
| `[]`     | Array subscript |
| `()`     | Function call |
