## Matrix Operations

### Matrix Multiplication
```HLSL
mul(InVector, InMatrix);
```

### Transpose
```HLSL
transpose(InMatrix);
```

### Determinant
```HLSL
determinant(InMatrix);
```

## Vector Operations

### Dot Product
```HLSL
dot(v1, v2);
```

### Cross Product
```HLSL
cross(v1, v2);
```

### Length (Magnitude)
```HLSL
length(v);
```

### Normalize
```HLSL
normalize(v);
```

### Subtraction
```HLSL
float result = a - b;
```

### Multiplication
```HLSL
float result = a * b;
```

### Division
